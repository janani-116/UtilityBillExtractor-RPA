# Utility Bill Data Extractor - RPA Mini Project

## Overview
An RPA bot built using UiPath that automates utility bill data extraction and sends email alerts for unpaid bills.

## Tools Used
- UiPath Studio Community Edition
- HTML (Mock Portal)
- Microsoft Excel
- Gmail SMTP

## Features
- Reads customer IDs from Excel
- Scrapes bill details from web portal
- Updates Excel with extracted data
- Sends email alerts for unpaid bills
